# museaigc



\# 安装环境要求



(1) 操作系统：Windows 11 64 位专业版

(2) 运行环境：Python 3.13.12 解释器

(3) 核心计算框架：PyTorch

(4) 客户端环境：Google Chrome、Microsoft Edge 等主流 Web 浏览器



\#  安装过程



1. 下载项目源代码并解压到本地目录。

2\. 打开命令行，进入项目根目录，执行以下命令安装依赖包： pip install -r requirements.txt

3\. 运行启动脚本streamlit run app.py

4\. 启动成功后，在浏览器中访问提示的本地地址即可进入系统。

5\. 安装HBuilderX开发工，启动移动端muse\_ai文件夹中的index.vue
